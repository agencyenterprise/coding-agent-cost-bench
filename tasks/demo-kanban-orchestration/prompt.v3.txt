Empty workspace: scaffold with `create-next-app` (App Router, TypeScript) and install only what you need. Add at least one Vitest test for the store's move-card logic. Before finishing, confirm `npm run build` and `npx vitest run` both pass (exit 0) and that `/api/board` returns 3 columns and 6 cards.
